Overview:

This is an updated version of the IE Fix Wizard. It is built from the ground up to be faster and easier to use. It is designed for Internet Explorer 11 users in Windows 7 or Windows 10.

We are looking for feedback from the pilot test group before we make this available to all Boeing users. If you are interested in testing the tool out for yourself, please click the link below.

The tool is still in a testing stage so if you have any feedback (positive or negative) you can email me at alexander.p.lied@boeing.com.

After running the IE Fix Wizard, a local log file is created in the location C:\temp\FixWizard.log. If you encounter an error I would appreciate it if you attach that file to your email to help me diagnose the problem.



The IE Fix Wizard is a utility that can be used to repair the configuration of Internet Explorer.
This revision of the wizard supports Internet Explorer 11 for Window 7 and Windows 10.
The utility detects the current platform and IE version to determine the correct configuration repair capabilities. The user can then use the appropriate repair option.

The updated fix wizard also validates certain standard configuration artifacts, displays a warning message and a repair option if these artifacts are not present.

This fix wizard cannot be used to upgrade IE between versions, new versions are available in the standard Software Express repository when they become available.

Usage:

  • click: IEFixWizard.exe
  • Select "Save" or "Save as..." when prompted
  • Save the application to your desktop or to the "downloads" folder
  • Double-click the file from the location you saved it

 

See the IE Fix Wizard FAQ for descriptions of the options that may be displayed by the Fix Wizard. Not all options are available for all configurations, the utility will display only those options applicable to the current platform. Each option should included a "More Info..." link that should open a web page with more information about that specific option.

In addition, the fix wizard may detect other configuration issues with the device that may affect the platform or IE. These warnings or errors will attempt to offer either a direct fix or provide additional guidance for how the issue can be resolved.

In normal operation IE Fix Wizard will look like this:

Options:

  • Main = main information panel, standard restoration option
  • Fixes = Potential fixes to be applied
    • specific fix options depend on the specific configuration
    Options = Advanced Options
    • Reset to Microsoft Defaults
    • Install Configuration Component (Applies configuration to the local machine as well as the current user)
